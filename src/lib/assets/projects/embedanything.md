I am the author and maintainer of EmbedAnything — a high-performance Rust-based library for multi-modal embedding pipelines with over **750 GitHub stars** and **300,000+ downloads**. It supports local, ONNX, and cloud embedding models, handles text, images, and audio, and uses vector streaming for memory-efficient indexing.

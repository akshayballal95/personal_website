An AI-powered engineering drawing analysis platform for manufacturing teams. Users upload PDF or image drawings and a LangGraph + Google Gemini pipeline extracts dimensions, tolerances, and metadata, with YOLO detecting annotation regions and automated machine capability fit checks. Built as a monorepo with a React/TypeScript frontend and Python FastAPI backend deployed on AWS Lambda.

[Live App](https://takshaai.triviz.cloud)

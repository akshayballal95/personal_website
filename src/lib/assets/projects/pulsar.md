A Cursor-like web IDE for knowledge work: documents, blog posts, research notes, and drafting alongside AI while staying in the loop. Three panels: file explorer, Monaco editor with markdown preview and PDF viewer, and a streaming agent chat with visible tool calls, diffs, threads, and Agent Skills. Built with Next.js and deployed with persistent agent sandboxes on AWS.

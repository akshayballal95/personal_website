A multi-platform desktop application that indexes local files and enables semantic search across them. Built with RAG techniques including hybrid search and reranking to improve retrieval quality. Uses Ollama for local LLMs and OpenAI/Gemini API for cloud models, with LanceDB as the local vector store.

[Website](https://www.starlight-search.com)
